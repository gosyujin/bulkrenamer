# BulkRenamer

## Requirements

- homebrew or go

## Install

### homebrew

```sh
$ brew tap gosyujin/bulkrenamer                                                                                                                                                     
$ brew install bulkrenamer
```

### go

```sh
$ go get github.com/gosyujin/bulkrenamer
```

## Usage

```sh
$ bulkrenamer DIR
```
